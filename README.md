# PolyRhythma
Practice polyrhythms on your keyboard. Custom-define rhythms and get real-time accuracy feedback.

## Quickstart
- Requirements: Node 18+ and npm.
- Install: `npm install`
- Develop: `npm run dev` then open the shown local URL.
- Build: `npm run build` (outputs to `dist/`).
- Preview production build locally: `npm run preview`.
- Deploy: Vite `base` is set to `/PolyRhythma/` for GitHub Pages; keep that path when hosting.
